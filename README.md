# 🎮 Games

한성대학교 네트워크프로그래밍 프로젝트용 미니 게임

---

## 포함된 게임 목록

### 1. ShootingGame  
- 키보드로 몬스터를 공격하는 슈팅 게임  
- 다양한 몬스터 스폰, 목숨 시스템, 점수 시스템 포함  
- 배경 이미지, BGM, 효과음 추가됨  

### 2. SwipeBreakoutGame  
- 벽돌깨기 게임  
- 공 튀김, 패들 이동, 점수 계산, 게임 오버/클리어 기능  
- 인게임 BGM & 효과음 포함  

### 3. TypingGame  
- 단어가 떨어지기 전에 타이핑해서 없애는 산성비 스타일 게임  
- 단어 파일(txt) 로드  
- 1P / 2P 멀티플레이 버전 지원  

### 4. VolleyGame  
- 피카츄 배구 느낌의 1:1 네트워크 대전 게임  
- 서버/클라이언트 구조  
- 스파이크, 점프, 공 물리엔진 구현  
- 배경음악 / 효과음 포함
